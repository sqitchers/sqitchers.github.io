Sane database change management

Sqitch is the the sane database change management system, emphasizing the strengths of native database tools and syntaxes, development iteration, and deployment integrity.

Inspired by Git Sqitch offers a familiar interface but no opions about languages or frameworks, with features like dependency resolution, mulitproject management, and deployment target management.
